Properties

There are few different background properties

background-image
This property is the location of the image itself,for example h1 { backgound-image:url('https://website.com/img/background.jpg')}
or

h1 { background-image: url('images/background.jpg')}
background-color
this allows us to add color on our background,an example h1 {background-color: green};
background-repeat
This is how the image can be repeat p.The value can be any of the following.
background-repeat: repeat;

This repeats on both x and y axis and will essentially file the image across the entire background


background-repeat: repeat-x

The repeats ther background on the x axis,or horizantally


background-repeat: repeat-y

The repeats ther background on the y axis,or vertically


background-repeat: no-repeat

this shows image only once without reoeating


Background-poisition
This is actual poisition of the backgroundthe image withen its container The values can be; background-poisition: top; background-poisition: right; background-poisition: bottom; background-poisition: left; background-poisition: center;
It can also be the percentage, length, or combination,like so:
background-poisition: 50px 20%;
